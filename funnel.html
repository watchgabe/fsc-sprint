YouTube
Not connected
▼
Your YouTube Data API v3 key
Found in YouTube Studio → Settings → Channel → Advanced
How to get these? →
  1. Go to Google Cloud Console
  2. Create a new project (or select existing)
  3. Enable "YouTube Data API v3" in the API Library
  4. Go to Credentials → Create Credentials → API Key
  5. Copy the API key and paste it above
  6. For Channel ID: open YouTube Studio → Settings → Channel → Advanced settings
Instagram
Not connected
▼
Use manual entry instead of API
How to get Instagram API access? →
  1. Go to Meta for Developers
  2. Create an app → choose "Consumer" type
  3. Add the Instagram Basic Display product
  4. Under User Token Generator, add your Instagram account
  5. Generate a User Token — this is your Access Token
  6. Your User ID appears alongside the token
  7. Note: If CORS blocks the API, the dashboard will use your manual data automatically
Kit
Not connected
▼
Found in Kit → Settings → Developer → API Keys
Manual Fallback (used if API unavailable)
How to get Kit API key? →
  1. Log in to Kit (formerly ConvertKit)
  2. Go to Settings → Developer
  3. Under "API Keys", click "Add API Key"
  4. Copy the key and paste it above
  5. Note: Kit's API may block browser requests (CORS). If so, enter your stats manually in the fallback fields above.
Circle
Not connected
▼
Manual Fallback
How to get Circle API token? →
  1. Log in to your Circle community
  2. Go to Settings → API
  3. Generate a new API token
  4. Copy and paste it above along with your community URL
  5. Note: Circle's API may require server-side calls. Use manual entry as a reliable fallback.
